# DSA Problems in C# 🚀

A comprehensive collection of Data Structures and Algorithms problems solved in C#, organized by patterns for easy reference and learning.

## 📂 Project Structure

```
DSA-CSharp/
├── AGENT_GUIDELINES.md          # ⚠️ READ THIS BEFORE CREATING ANY PROBLEM
├── README.md                     # This file
├── PATTERN_INDEX.md              # Pattern reference guide
├── QUICK_START.md               # Quick start guide
├── Arrays/                       # Array-based problems
├── Stacks/                       # Stack-based problems
├── Graphs/                       # Graph algorithms
└── [More folders created as problems are added]
```

**Note**: Pattern folders are created progressively as problems are added.
See AGENT_GUIDELINES.md for the complete list of available patterns.

## 🎯 Purpose

This repository serves as:
- **Study reference** for DSA interview preparation
- **Pattern-based organization** for efficient learning
- **Optimal solutions** with detailed explanations
- **Complexity analysis** for every problem

## 📝 Problem Format

Each problem includes:
- ✅ **Clear problem statement**
- ✅ **Input/Output examples**
- ✅ **Constraints**
- ✅ **Edge cases** - Common boundary conditions
- ✅ **Trick cases** - Gotchas and pitfalls
- ✅ **Optimal solution** - Best known approach
- ✅ **Simple explanation** - Easy to understand
- ✅ **Time & Space complexity** - Big O analysis
- ✅ **Pattern identification** - What technique is used

## 🚀 Quick Start

1. Browse problems by pattern folder
2. Open a problem file to see structure
3. Read the problem statement and constraints
4. Review the optimal solution explanation
5. Study the implementation

## 📚 Learning Path

Recommended order by pattern:
1. **Arrays & Strings** - Foundation
2. **HashMaps** - Fast lookups
3. **Two Pointers / Sliding Window** - Optimization techniques
4. **Stacks & Queues** - Linear data structures
5. **LinkedLists** - Pointer manipulation
6. **Trees** - Recursive thinking
7. **Graphs** - Advanced traversal
8. **Dynamic Programming** - Optimization problems
9. **Backtracking** - Constraint satisfaction
10. **Advanced patterns** - Heaps, Tries, etc.
